[![Stars](https://flat.badgen.net/github/stars/ripred/.github)](https://github.com/ripred/.github/stargazers)
[![Forks](https://flat.badgen.net/github/forks/ripred/.github)](https://github.com/ripred/.github/network/members)

This repository provides fallback community-health files for ripred repositories.

GitHub uses these files when a repository does not define its own versions:

- Code of conduct
- Contributing guide
- Security policy
- Support guide
- Issue forms
- Pull request template

Repository-specific files take precedence over these defaults.

Licenses are intentionally not provided here. Each repository needs its own license decision because the projects are not all the same type of work.
